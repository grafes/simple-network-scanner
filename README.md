# Simple Network Scanner

## Description
This is a simple Python-based network scanner that detects open ports on a target IP address.

## Features
- Scan custom port ranges
- Detect open ports
- Command-line interface

## Technologies Used
- Python
- Socket

## How It Works
The program attempts to connect to each port in the given range.
If a connection is successful, the port is marked as open.

## Usage
```bash
python3 scanner.py

Example

Target IP: 127.0.0.1
Start Port: 1000
End Port: 2000

Output

[OPEN] Port 1234

Project Purpose

This project was created to learn basic networking concepts and Python socket programming.
